# Roadmap

## v0.2：接入大模型 API

- 基于 RAG 检索结果生成更自然的中文回答。
- 基于证据链生成更完整的复核报告。
- 对低置信度和冲突证据给出更清晰的问题追问。

## v0.3：RAG 评估集

- 构建标准测试 query 集。
- 统计 top-k 命中率。
- 统计证据来源类型命中准确率。
- 对强依据、中等依据、弱线索分别评估召回效果。

## v0.4：文件上传和增量建库

- 支持在页面上传 Excel、CSV、Markdown 和 PDF。
- 支持增量解析和增量入库。
- 支持对不同资料版本做来源追踪。

## v0.5：人工复核流

- 为每条变更增加人工复核状态。
- 支持确认、退回、补证、暂缓纳入等状态流转。
- 支持导出复核后的正式候选变更清单。
